# ‚úÖ SCENARIO 1 ‚Äì Windows 10 System Investigation & Hardening

## Q1. Identify two suspicious processes or services

### Step 1: Open Task Manager / Services

* Press **Ctrl + Shift + Esc**
* Go to **Processes** and **Services** tabs

### Step 2: Identify suspicious behavior

Examples:

1. **Unknown executable**

   * Process name: `svch0st.exe` (note: legitimate is `svchost.exe`)
   * Location: Running from `C:\Users\Public\` instead of `C:\Windows\System32`

2. **Suspicious service**

   * Random name (e.g., `WinUpdateService123`)
   * Startup type: **Automatic**
   * No description or vendor info

### Step 3: Why suspicious?

* Legitimate system services:

  * Have **Microsoft Corporation** as publisher
  * Run from **System32**
* Unknown names + wrong path = **possible malware persistence**

## Q4. Verify BitLocker

### Step 1: Check status

* Control Panel ‚Üí BitLocker Drive Encryption
* Or:

```powershell
manage-bde -status
```

### Step 2: Enable BitLocker (if disabled)

1. Turn on BitLocker
2. Choose TPM / password
3. Save recovery key
4. Encrypt entire drive

### Step 3: Importance

* Protects data if:

  * System stolen
  * Disk removed
* Prevents offline access to sensitive files
